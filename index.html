Country Site

This project is a dynamic, interactive web application built with React, React Router V7, TypeScript, and Tailwind CSS. It fetches country data from the REST Countries API, allowing users to explore and filter countries by name and region. The application uses dynamic routing to display detailed information about each country, including its name, capital, population, region, and flag. With a responsive design powered by Tailwind CSS, the app provides a seamless user experience across devices. The code is modular and organized into reusable components, ensuring scalability and ease of maintenance. Vite is used for fast development and efficient builds.

React ChatBot

I developed a React-based chatbot that integrates with the OpenAI API to provide intelligent conversational capabilities. Using React, Tailwind CSS, and Axios, I built a responsive and user-friendly interface, with dynamic styling powered by Tailwind. This project demonstrates my skills in modern web development, API integration, and creating seamless user experiences.

Next E-commerce

I developed an e-commerce platform that integrates Stripe for seamless payment processing and features real-time cart updates. This project showcases my ability to integrate payment systems, manage state effectively, and build scalable e-commerce solutions.
